# Sakaar Interior - Website

## Setup Instructions

1. Save your logo image as `logo.png` in the same folder as index.html
2. Open `index.html` in your browser to view the website

## Logo Requirements
- The logo should be saved as `logo.png`
- Recommended size: 200x200px or similar square/rectangular format
- The logo will be automatically resized to fit the navbar (50px height on desktop, 40px on mobile)
